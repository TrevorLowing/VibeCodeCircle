# EtchWP Compatibility Review

This directory contains EtchWP plugin and theme files for compatibility review with vibecode-deploy.

## Files Expected

1. `etchwp-plugin.zip` - Current EtchWP plugin zip file
2. `etch-theme.zip` - Current Etch theme zip file

## Extraction Locations

- Plugin: `extracted/etchwp-plugin/`
- Theme: `extracted/etch-theme/`

## Purpose

Review EtchWP plugin and theme to ensure vibecode-deploy creates content that is:
- Visible in EtchWP editor
- Compatible with EtchWP template system
- Properly stored (post_content vs templates)

## Review Output

Findings will be documented in `COMPATIBILITY_REPORT.md` in this directory.
